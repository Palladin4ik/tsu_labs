# Лабораторная работа 1:

## Использование 1 программы
1. Перейдите в каталог lab_rab_1
```bash
cd ./lab_rab_1
```
2. Сделайте файл 1_1.py исполняемым:
```bash
chmod +x ./1_1.py
```
3. Запустите файл:
```bash
./1_1.py 2> error.txt
```
